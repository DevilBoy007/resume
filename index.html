hex colors

hex colors are a hexadecimal representation of a color.

they are used in web development (and other places) to define the color of an element.

the format is a 6-digit string that represents the red, green, and blue values of the color.
the first two digits represent the red value, the second two represent the green value, and the last two represent the blue value.
the values range from 0 to 255 and are represented as base 16 [1016] numbers.
